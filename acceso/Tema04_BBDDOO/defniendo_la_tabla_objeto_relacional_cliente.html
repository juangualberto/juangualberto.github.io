Defniendo la tabla objeto relacional cliente

Una tabla objeto relacional se define a partir de los objetos que hemos visto anteriormente. Así el objeto Cliente (la "clase" cliente) podemos usarla como esquema para definir una tabla de este modo:

CREATE TABLE Clientes_objtab OF Cliente_objtyp (Cliente PRIMARY KEY) OBJECT ID PRIMARY KEY;
