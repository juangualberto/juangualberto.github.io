Excepciones en nuestros componentes

Para crear nuestras propias excepciones, deberemos tener una clase dedicada a ello.

Ejemplo de excepción para una API de nuestra creación:

/**
 *
 * @author Juangu &lt;jgutierrez at iesvirgendelcarmen.com&gt;
 */
public class ExcelApiException extends Exception {

    /**
     * Creates a new instance of <code>ExcelUtilException</code> without detail
     * message.
     */
    public ExcelApiException() {
    }

    /**
     * Constructs an instance of <code>ExcelUtilException</code> with the
     * specified detail message.
     *
     * @param msg the detail message.
     */
    public ExcelApiException(String msg) {
        super("ExcelAPI: "+msg);
    }
}

Ejemplo de cómo usar esta Excepción:

   /**
     * 
     * @return the number of sheets in this WorkBook
     * @throws ExcelApiException if the DataSheets contained were never initilized
     */
    public int getNumberOfSheets() throws ExcelApiException {
        int returnValue = -1;
        if (this.sheets!=null){
            returnValue=this.sheets.size();
        } else {
            throw new ExcelApiException("WorkBook.getNumberOfSheets():: Null pointer...");
        }
        return returnValue;
    }
